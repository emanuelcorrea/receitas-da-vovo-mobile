### Installing

**Cloning the Repository**

```
$ git clone https://github.com/emanuelcorrea/receitasdavovo-mobile.git

$ cd receitasdavovo-mobile
```

### Running

After to downloaded or clonned the repositore, you can now run the app:

Android

```
$ flutter run
```
